## Install dependencies

(optional) Install PNPM [here](https://pnpm.io/).

```bash
pnpm install
```

### Local dev

Runs all build:css, typecheck and local dev server

```bash
pnpm start:dev
```

or run scripts seperately

```bash
pnpm typecheck
```

```bash
pnpm watch:css
```

```bash
pnpm dev
```

finally do a sanity with vercel dev

```bash
vercel dev
```
