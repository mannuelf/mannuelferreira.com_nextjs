## Observer pattern

Defines a one-to-many dependency between objects that when on eobject changes state, all of its dependents are notified and updated automatically.

Observer

```python
from abc,  import ABC, abstractmethod

class Observer(ABC):
    @abstractmethod
    def update(self, message):
      pass
```

Publisher

```python
from abc,  import ABC, abstractmethod

class Newspaper(ABC):
    @abstractmethod
    def add_subscriber(self, subscriber: Observer):
      pass

    @abstractmethod
    def remove_subscriber(self, subscriber: Observer):
      pass

    @abstractmethod
    def notify_subscriber(self, subscriber: Observer):
      pass
```

Concrete Observer

```python
class Subscriber(Observer):
    def __init__(self, name):
        self.name = name

    def update(self, message: str):
      print(f"{self._name} received the message {message}")
```

Concrete Subject

```python
class OnlineNewsPaper(Newspaper):
    def __init__(self, name: str):
      self._name = name
      self.subscriber = set()

    def publish_article(self, title):
      self.notify.subscribers(f"New article posted with title {title}")

    def add_subscriber(self, subscriber: Observer):
      self.subscribers.add(subscriber)

      self.subscribers.remove(subscriber)
```
